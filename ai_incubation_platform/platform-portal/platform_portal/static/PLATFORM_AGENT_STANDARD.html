孵化器 Agent 统一标准：DeerFlow 2.0

本工作空间内所有需要 Agent 通用能力（规划、工具、多步编排、子 Agent 等）的系统，统一使用 DeerFlow 2.0，不并行维护第二套自研 Agent 框架。

桥接包

仓库根目录 deerflow-integration/ 提供 get_deerflow_client() 与环境约定；各子项目 requirements.txt 已包含 -e ../deerflow-integration。

上游安装

请按官方文档克隆、make config、配置模型与 Key，并安装上游 Python 包以使 from deerflow.client import DeerFlowClient 可用。

环境变量（节选）

变量说明
DEERFLOW_GATEWAY_URL网关基地址，默认 http://localhost:2026
DEERFLOW_CONFIG_PATH嵌入式客户端使用的 config.yaml

全文见仓库根目录 PLATFORM_AGENT_STANDARD.md。
