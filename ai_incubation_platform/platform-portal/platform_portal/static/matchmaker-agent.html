🧮 匹配算法

多维度加权综合匹配度计算：

综合匹配度 = 兴趣相似度 × 0.3 + 价值观相似度 × 0.35 + 年龄适配度 × 0.15 + 地点适配度 × 0.2

🔌 API 端点

  • POST /api/users/register 用户注册
  • POST /api/users/login 用户登录 (JWT)
  • GET /api/users 获取用户列表
  • GET /api/users/{id} 获取用户详情
  • PUT /api/users/{id} 更新用户 (需认证)
  • GET /api/matching/user/{id}/matches 获取推荐匹配
  • POST /api/matching/compare 比较两人匹配度
