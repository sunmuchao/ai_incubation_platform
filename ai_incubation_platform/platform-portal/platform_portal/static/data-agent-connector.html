💬 NL2SQL 演示

自然语言: "查询所有用户"
→ SELECT * FROM users
自然语言: "查询年龄大于 18 的用户"
→ SELECT * FROM users WHERE age > 18

🔌 API 端点

  • POST /api/query/connect 连接数据源
  • GET /api/query/connectors 连接器列表
  • POST /api/query/execute 执行查询
  • POST /api/query/nl2sql 自然语言查询
  • GET /api/query/schema 获取数据结构
