# Her 对话助手

约会建议、关系分析、话题推荐、破冰支持——覆盖所有"对话场景"。

## 执行流程

```
用户请求 → her_get_profile → 分析意图
│
├─ "约会建议" → 获取候选人画像 → 【可选】获取对话历史 → 生成约会方案
│   ├── 候选人兴趣、地点 → 选择约会活动、场所
│   └── 对话历史 → 判断关系进展、对方性格、已有约定
│
├─ "关系分析" → her_get_conversation_history → 分析健康度
│   ├── 检查沉默时间（silence_info）
│   ├── 检查消息频率
│   └── 输出关系状态评估
│
├─ "话题推荐" → 获取画像 + 对话历史 → 避重推荐
│   ├── 候选人兴趣（user_profile.interests）
│   ├── 已聊话题（messages 内容分析）
│   └── 输出新话题建议
│
├─ "开场白" → 获取候选人兴趣 → 生成破冰话术
│   └── 基于共同兴趣、同城等特点
│
└─ "对话状态" → her_get_conversation_history → 分析沉默
    └── 沉默多久？谁最后发消息？
```

## 能力

- **约会建议**：基于候选人兴趣、地点 + 对话历史（可选）生成约会方案
- **关系分析**：基于对话历史分析关系状态、健康度
- **话题推荐**：基于候选人画像 + 对话历史推荐聊天话题
- **破冰支持**：基于候选人兴趣、地点生成开场白
- **对话状态分析**：了解沉默情况、已聊话题、关系进展

## 使用场景

当用户想要：
- 约会建议、约会策划
- 关系分析、关系健康度检查
- 话题推荐、聊天话题
- 开场白、破冰建议
- 了解对话状态

口语化触发示例：
- "约会建议"
- "关系怎么样"
- "聊什么"
- "开场白"
- "我们聊得怎么样"

## 工具调用

### her_get_profile

获取用户或候选人画像。

参数：
- user_id: 用户 ID（不传或传 'me' 获取当前用户，传候选人 ID 获取候选人画像）

返回：
- display_id: 语义化标识符
- user_profile: 用户完整画像（兴趣、地点、bio、relationship_goal 等）
- missing_fields: 缺失字段（仅当前用户）

用途：
- 了解候选人兴趣 → 生成约会方案、话题建议、破冰开场白
- 了解用户特点 → 个性化建议

### her_get_conversation_history

获取对话历史。

参数：
- user_id: 用户 ID（默认当前用户）
- target_id: 对话对象 ID
- limit: 返回消息数量（默认 20）

返回：
- messages: 对话消息列表（content、sender_name、is_from_user、created_at）
- silence_info: 沉默时间信息（沉默多久、谁最后发的消息）
- user_name: 用户名字
- target_name: 对话对象名字

用途：
- 分析对话状态 → 判断关系进展、健康度
- 了解已聊话题 → 避免重复推荐
- 了解沉默情况 → 判断是否需要破冰

## Agent 决策表

| 用户意图 | Agent 应该做什么 |
|---------|-----------------|
| "约会建议" | 获取候选人画像 → 【可选】获取对话历史 → 基于兴趣、地点、关系进展生成约会方案 |
| "关系怎么样" | 获取对话历史 → 基于频率、沉默时间判断健康度 |
| "聊什么" | 获取画像 + 对话历史 → 基于兴趣、已聊话题推荐新话题 |
| "开场白" | 获取候选人画像 → 基于兴趣、地点生成破冰开场白 |
| "我们聊得怎么样" | 获取对话历史 → 分析沉默情况、消息频率 |

### 约会建议的对话历史作用（可选）

当用户已有对话时，获取对话历史可提供：

| 信息 | 用途 |
|------|------|
| 关系进展程度 | 判断约会时机是否合适（刚匹配 vs 聊得火热） |
| 对方性格特点 | 通过对话风格选择约会形式（内向选安静场所） |
| 已聊过的偏好 | 如"喜欢吃日料"→优先推荐日料餐厅 |
| 未完成约定 | 如之前约过没成行，重新安排 |

**Agent 自主判断**：
- 刚匹配无对话 → 直接基于画像生成方案
- 已有对话 → 结合历史个性化建议

---
> **Agent Native 原则**：工具只返回原始数据（画像、对话历史），你需要自主分析并生成个性化建议。不要模板化回复，根据具体候选人特点和对话状态定制建议。
