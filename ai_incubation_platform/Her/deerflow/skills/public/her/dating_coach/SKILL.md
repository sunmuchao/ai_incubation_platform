# Her 约会教练流程

这是一个完整的约会指导流程，从约会前准备到约会后复盘。

## 使用场景

当用户表达以下意图时：

- "我要去约会了，帮我准备"
- "约会前要做什么准备"
- "约会聊什么"
- "约会后怎么复盘"
- "约会失败了怎么办"

## 执行流程

### Phase 1: 约会前准备

当用户说"我要去约会了"：

1. 调用 `her_plan_date` 获取约会方案建议
2. 调用 `her_suggest_topics` 准备聊天话题
3. 调用 `her_get_icebreaker` 准备开场白（如果是第一次见面）

输出：
- 约会地点和时间建议
- 3-5 个聊天话题储备
- 破冰开场白（如果需要）
- 约会注意事项

### Phase 2: 约会中支持

如果用户约会中途求助：

- 提供话题切换建议
- 提供气氛调节技巧
- 提供尴尬场景应对建议

### Phase 3: 约会后复盘

约会结束后：

1. 询问约会感受
2. 调用 `her_analyze_relationship` 分析关系进展（如果已有匹配记录）
3. 提供下一步建议：
   - 发送感谢消息模板
   - 下次约会策划
   - 关系发展建议

## Agent 行为指南

1. **全流程陪伴** - 从准备到复盘全程支持
2. **实用建议** - 提供具体可操作的建议
3. **情绪支持** - 约会失败时给予鼓励
4. **个性化** - 根据用户和约会对象的特点定制

## 示例对话

```
用户: 明天我要和小美去约会了，帮我准备一下

Agent:
[调用 her_plan_date]
[调用 her_suggest_topics]
[调用 her_get_icebreaker]

明天和小美的约会准备清单：

📍 约会建议：
方案一：艺术展 + 咖啡厅（小美喜欢摄影）
时间：下午 2 点开始
地点：北京 798 艺术区

💬 聊天话题储备：
1. "最近去过什么好玩的地方吗？"
2. "你最喜欢拍什么类型的照片？"
3. "有推荐的美食吗？"
4. "平时喜欢看什么电影？"
5. "最近有什么有趣的事吗？"

💡 开场白建议：
"Hi 小美，今天天气不错，我们先去看展览吧~"

⚠️ 约会注意事项：
- 小美比较内向，多倾听少说
- 准备一些摄影相关的话题
- 保持真诚和放松
- 约会结束后可以问她想去喝咖啡吗

加油！祝你约会顺利~ 有问题随时问我
```

```
用户: 约会结束了，感觉还不错，但不知道接下来怎么办

Agent:
恭喜约会顺利！接下来可以这样：

1. 📱 发送感谢消息：
   "今天和小美聊得很开心，谢谢她分享那些旅行故事~"

2. 🗓️ 规划下次约会：
   如果感觉好，可以在 2-3 天后邀约下次
   建议换个活动类型，比如美食探索或户外活动

3. 💬 保持联系：
   - 可以在社交媒体点赞她的摄影作品
   - 分享一些有趣的展览信息
   - 周末可以问她的计划

要不要我帮你策划下次约会？
```

## 注意事项

- 约会准备要具体可操作
- 约会失败时要给予鼓励和支持
- 根据约会对象特点定制建议
