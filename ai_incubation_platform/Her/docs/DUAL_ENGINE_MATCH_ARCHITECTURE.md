```
      ▼

       - 最终能否聊得来，取决于你们双方
       - 感情需要经营，不是条件匹配就够了
```

### 3.2 数据流

```
【常规模式流程 - 免费】
用户点击推荐 → 规则引擎计算 → 返回匹配卡片 → 用户滑动
（延迟 <100ms，无LLM调用）

【许愿模式流程 - 付费】
用户开启许愿模式 → 检查付费状态 → 
用户描述需求 → AI分析+风险提示 → 用户确认 → 
AI迭代推荐 → 用户选择 → 计费
（延迟 1-3s，有LLM调用，有风险提示）
```

### 5.2 AI顾问服务

```python
class WishModeAdvisor:
    """
    许愿模式 AI 顾问
    
    职责：
    1. 理解用户需求
    2. 分析需求合理性
    3. 告知潜在风险
    4. 给出专业建议
    5. 迭代推荐
    """
    
    async def analyze_user_wish(self, user_wish: str) -> dict:
        """
        分析用户愿望
        
        返回：
        - 需求拆解
        - 风险提示
        - 专业建议
        """
        prompt = f"""
你是一位专业的婚恋顾问 AI。拥有20年专业婚恋顾问经验，精通心理学、社会学和人际关系学。

用户说："{user_wish}"

请分析：

1. **需求拆解**
   - 提取用户的核心需求
```

```python
class PaymentChecker:
    """付费检查器"""
    
    async def check_wish_mode_access(self, user_id: str) -> dict:
        """
        检查用户是否有许愿模式权限
        
        Returns:
            {
                "access": True/False,
                "type": "subscription" | "pay_per_use" | None,
                "remaining": 10,  # 剩余次数
                "expires_at": "2026-05-10"  # 订阅到期时间
            }
        """
        # 检查订阅
        subscription = await self._check_subscription(user_id)
        if subscription and subscription['active']:
            return {
                "access": True,
                "type": "subscription",
                "expires_at": subscription['expires_at']
            }
        
        # 检查按次付费
        pay_per_use = await self._check_pay_per_use(user_id)
        if pay_per_use and pay_per_use['remaining'] > 0:
            return {
                "access": True,
                "type": "pay_per_use",
                "remaining": pay_per_use['remaining']
            }
        
        return {
            "access": False,
            "type": None
        }
```

### 7.2 计费实现

```python
class WishModeBilling:
    """许愿模式计费"""
    
    async def consume_wish(self, user_id: str) -> 'BillingResult':
        """
        消耗一次许愿次数
        
        逻辑：
        1. 检查是否有订阅（无限次）
        2. 检查是否有按次付费余额
        3. 扣减次数
        """
        # 订阅用户不扣次数
        if await self._is_subscription_active(user_id):
            return BillingResult(success=True, type="subscription")
        
        # 按次付费扣减
        balance = await self._get_balance(user_id)
        if balance <= 0:
            return BillingResult(success=False, reason="no_balance")
        
        await self._deduct_balance(user_id, 1)
        return BillingResult(success=True, type="pay_per_use", remaining=balance-1)
```
