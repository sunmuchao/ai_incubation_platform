## 架构对齐验证

### DeerFlow 2.0 标准架构

```
┌─────────────────────────────────────────────────────────┐
│                    platform-portal                       │
│                    (统一入口)                             │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                  agent-platform-core                     │
│                  (DeerFlow 2.0 核心)                      │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐       │
│  │ DeerFlow    │ │ Tools       │ │ Audit       │       │
│  │ Client      │ │ Registry    │ │ Logger      │       │
│  └─────────────┘ └─────────────┘ └─────────────┘       │
└─────────────────────────────────────────────────────────┘
                          │
    ┌─────────────┬───────┼───────┬─────────────┐
    ▼             ▼       ▼       ▼             ▼
┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐
│ ai-hires  │ │ community │ │ employee  │ │ opportunity│
│  - Task   │ │  - Group  │ │  - Talent │ │  - Miner  │
│   Agent   │ │   Agent   │ │   Agent   │ │   Agent   │
└───────────┘ └───────────┘ └───────────┘ └───────────┘
```

### 统一架构模式

所有 12 个项目均实现了以下架构层：

| 层级 | 文件 | 说明 |
|------|------|------|
| **Agents** | `src/agents/__init__.py` | Agent 层入口 |
| | `src/agents/deerflow_client.py` | DeerFlow 客户端 |
| | `src/agents/*_agent.py` | 领域 Agent |
| **Tools** | `src/tools/__init__.py` | Tools 层入口 |
| | `src/tools/*_tools.py` | 领域工具集 |
| **Workflows** | `src/workflows/__init__.py` | Workflows 层入口 |
| | `src/workflows/*_workflows.py` | 声明式工作流 |
| **API** | `src/api/chat.py` 或 `src/api/ai_native.py` | 对话式 API |
| **文档** | `AI_NATIVE_REDESIGN_WHITEPAPER.md` | 白皮书 |
| | `AI_NATIVE_COMPLETION_REPORT.md` | 完成报告 |
