# AI 社区团购 — 规划

## 长期愿景

面向社区场景的 **AI 驱动供需匹配**：在选品、成团概率与定价上辅助团长决策，降低运营摩擦，并支撑小规模电商闭环（商品—团购—履约）。

## 路线图（当前）

### P0 — 高优先级

- 商品/库存/团购单/参团最小业务闭环；成团失败、过期与库存扣减规则在数据模型中明确。
- API 与 README 目标能力对齐，可本地端到端演示。

### P1 — 中优先级

- 选品与定价策略（可先规则引擎，再接入 LLM 辅助说明）。
- 库存预警与基础通知占位。

### P2 — 较低优先级

- 地域/社群维度运营能力；与统一认证集成。
